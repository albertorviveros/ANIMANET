APP ANDROID ANIMANET

URL configurada:
https://www.animanet.org/animanet/

Cómo compilar:
1. Abre esta carpeta en Android Studio.
2. Espera a que sincronice Gradle.
3. Conecta un teléfono o usa emulador.
4. Presiona Run para probar.
5. Para generar APK: Build > Build Bundle(s) / APK(s) > Build APK(s).

Funciones incluidas:
- WebView con JavaScript y cookies.
- Abre WhatsApp, teléfono y correo fuera de la app.
- Soporte para descargas.
- Pantalla de sin conexión.
- Botón atrás navega dentro del sistema.
